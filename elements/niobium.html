   Niobium, formerly columbium, is a chemical element with symbol nb(formerly Cb) and atomic number 41. It is soft, grey, ductile transition metal, which is often found in the pyrochlore mineral, the main commercial source for niobium, and columbite. The name comes from Greek mythology:Niobe, daughter of Tantalus since it is so similar to tantalum.
